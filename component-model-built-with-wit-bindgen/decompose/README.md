# 组件解析

## 项目结构说明

文件 | 说明
-----|-----------
app | 应用外部包 docs:adder@0.1.0 的组件
cli | 解析 app 打包所得 WASM 组件的命令行工具

## 快速开始

```bash
make run
```

> 具体涉及的操作参见 [Makefile](./Makefile)。
